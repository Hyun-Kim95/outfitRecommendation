# 문서 인덱스

프로젝트 문서는 역할별 하위 폴더로 나뉜다.

| 폴더 | 용도 |
|------|------|
| [`requirements/`](requirements/README.md) | PRD, 리서치, 프론트/백엔드 계획 |
| [`design/`](design/README.md) | UI 명세, Stitch SOP |
| [`qa/`](qa/README.md) | 게이트 체크리스트, 단계별 QA 메모 |
| [`decisions/`](decisions/README.md) | 설계·정책 결정(ADR 스타일) |
| [`changelog/`](changelog/README.md) | 변경 이력 메모 |
| [`obsidian/`](obsidian/README.md) | 옵시디언 동기·대시보드 키 |

빠른 링크: [PRD](requirements/prd.md) · [UI 명세](design/ui-spec.md) · [게이트 체크리스트](qa/gate-checklist.md) · [테스트 리포트](../reports/test-report.md)
